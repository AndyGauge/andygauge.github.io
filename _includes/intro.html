MCP Authorization

Author of the mcp_authorization libraries across five languages — Ruby, Node, Python, Rust, and .NET — giving MCP servers a consistent, ergonomic auth layer in whichever stack the team already runs. In Ruby, it compiles inline RBS annotations into foldable schemas that replace JSON Schema, and is backed by a formal noninterference guarantee.

Sentinel RB

sentinel-rb is a system-level alternative to rbs-inline — a Rust-powered background watcher that keeps Ruby .rbs signatures in sync from inline annotations without interrupting your flow. It's fast enough to run as a pre-commit hook, asserting signatures are regenerated before code ever leaves your machine.

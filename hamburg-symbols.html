HamburgSymbols is a public-domain TrueType font built primarily for Uranian astrology. Each ASCII keystroke maps to a chart glyph — when text is set in this font, you get symbols. Type Q=12;42'53"f in HamburgSymbols and you read Q=12;42'53"f — Sun at 12°42′53″ Cancer.

How the notation works

Chart positions follow degree-minute-second form, with the planet on the left and the sign on the right. The middle is location.

  • ; renders as ; — the degree mark °
  • ' renders as ' — minutes
  • " renders as " — seconds
  • = renders as = — equals (used in midpoint equations)
  • @ renders as @ — slashed zero (some prefer it for whole-degree positions)

Midpoint equations look like U/W=Q (Saturn/Moon = Sun), typed as U/W=Q. Subscripts after a planet flag what kind of position it is: # natal, $ radix, % solar arc, & transit, ^ progressed, * directed. So U&dW# is "transiting Saturn directed to natal Moon" — typed as U&dW# (the d in between is just the letter directed-to; in this font that ASCII d renders as Gemini, so use it deliberately or replace with another connector).

A worked example

The PDF gives W=14;35'42"a — Moon at 14°35′42″ Aries. Type W=14;35'42"a. To layer in a midpoint, the format is planet/planet=planet:

U/W=Q

…typed U/W=Q. Saturn over Moon equals Sun — a classic Hamburg shorthand for "the Sun sits on the Saturn/Moon midpoint."

Using it elsewhere on the site

Anywhere on this site you can wrap text in <span class="font-hamburg">…</span> and the ASCII inside renders in HamburgSymbols. That works in Markdown posts too — the class is registered globally.
